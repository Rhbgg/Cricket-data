# Cricket Analytics Dashboard 🏏

An end-to-end data analytics project exploring ball-by-ball international cricket data across ODI, T20I, and Test formats.

## Project Overview
This project analyses over 1 million balls of international cricket data (2005–2020) to uncover insights on player performance, team strategies, and match patterns.

## Key Insights
- Top ODI run scorers and corrected batting averages
- Wicket takers ranked by wickets, economy and average
- Team win rates across 27 nations
- ODI scoring curve — how run rates evolve across 50 overs
- Dismissal type breakdown

## Tools & Technologies
- **Python** — data processing and analysis
- **Pandas** — data wrangling and aggregation
- **Google Colab** — cloud-based notebook environment
- **Chart.js** — interactive data visualisation
- **Streamlit** — dashboard framework (coming soon)
- **GitHub** — version control and project hosting

## Dataset
- Ball-by-ball data for ODI, T20I and Test matches
- Source: ESPNcricinfo (via structured dataset)
- Coverage: 2005–2025
- Size: ~1.1GB across 3 files
